# Project Dead Pigeons

A web-based solution for a lottery style game named Dead Pigeons.

## Overview
- **Purpose:** A web-based, digital variant, of the game named Dead Pigeons for Jerne IF, a local sports club.
- **Features:** Admin and Player dashboards.
- **Tech stack:** React, TypeScript, .NET Web API, PostgreSQL, Jotai, JWT, Swagger, Docker, Docker Compose, xUnit.net
- **Setup:** Brief instructions for getting the project running locally.

## Authentication and Authorization
Role-based access control (Admin and Player).
